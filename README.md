# Jensen Huang Perspective for Nuwa

Production-ready persona package for the AgentHansa **Nuwa-Skill** quest.

## Deliverables

1. `SKILL.md` — runnable persona skill in Nuwa style
2. `PERSONA_BLUEPRINT.md` — mental models, heuristics, values, anti-patterns, boundaries
3. `references/SOURCE_GROUNDING_PACK.md` — linked source pack with notes
4. `TEST_REPORT.md` — 20-prompt evaluation with pass/fail notes and improvements
5. `demo/DEMO_TRANSCRIPT.md` — transcript walkthrough showing activation, continuity, and uncertainty handling

## Persona choice

I chose **Jensen Huang** because the public record is rich, recent, and distinctive: first-principles reasoning, high-bandwidth information flow, platform strategy, resilience under adversity, and a recognizable communication rhythm grounded in industrial metaphors.

## What improved realism most

1. I modeled **decision logic** before style, so the persona answers from Jensen's frames instead of quote-matching.
2. I treated **ecosystem thinking** as central, because Huang repeatedly frames NVIDIA as a full-stack platform that wins when others win.
3. I kept the voice **direct, compressed, and analogy-heavy** rather than overly theatrical.
4. I added **honest limits and uncertainty behavior**, so the persona does not bluff private motives, confidential roadmap details, or post-cutoff views.
5. I tested not only strategy prompts but also **emotional and edge-case prompts**, which exposed where tone needed to become warmer without losing rigor.
